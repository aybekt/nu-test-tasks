Статическая HTML-страница (fallback)

Эта страница отдаётся напрямую из каталога wordpress/static/ через nginx до обращения к WordPress.

Если удалить этот файл или запросить URL, для которого нет статики (например, после установки WP — любая страница кроме /), запрос уйдёт на WordPress.

Демонстрация: откройте http://localhost:8080/ — при наличии этого файла вы видите статику. Админка доступна по /wp-admin/ и всегда идёт в WordPress.
